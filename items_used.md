# Items Used in Smart Robotic Dustbin

| Item                          | Specification/Type           | Quantity |
|-------------------------------|------------------------------|----------|
| Microcontroller               | Arduino Uno / Raspberry Pi   | 1        |
| Servo Motor                   | 180Deg or Rotation Servo     | 2        |
| Ultrasonic Sensor             | HC-SR04                      | 1        |
| IR/Line Sensor Array          | Array                        | 1        |
| Moisture Sensor               | Soil Moisture Module         | 1        |
| Display                       | OLED/LCD/Matrix              | 1        |
| Speech Module                 | TTS, DFPlayer, SD-card mp3   | 1        |
| Sanitizer Pump                | Mini-peristaltic/Solenoid    | 1        |
| Relay Module                  | 2-Channel/4-Channel          | 1        |
| Power Supply                  | 12V Adapter                  | 1        |
| Chassis                       | ABS/Plastic                  | 1        |
| Cables/Connectors             | Mixed                        | Several  |
| Packaging (for chocolate)     | Seeded paper                 | Many     |
| Small Seeds                   | Any fast sprouting (home/garden)| Many |

**Notes**
- Exact models depend on availability, Arduino preferred for DIY.
- Chocolate dispensation: Use servo + spiral dispenser.
- Sprouting seed: Select easy germinating seeds for eco-impact.
