# Smart Robotic Dustbin

A professional-grade robotic dustbin project. The robot:
- Opens lid automatically for waste
- Sorts waste: wet, dry, paper, covers
- Gives eco-reward (paper seeded chocolate)
- Dispenses sanitizer via hand gesture
- Shows daily quotes on display
- Expressive display with emotions
- Talks friendly to users

## Features
- **Waste Sorting**: Auto-separates wet, dry, paper, covers
- **Eco-Reward**: Dispenses chocolate wrapped in paper with a plant seed
- **Sanitizer Dispenser**: Touchless sanitizer via robot's hands
- **Quote Generator**: Shows daily quotes
- **Expressive Display**: Face/emoji expressions
- **Speech Interaction**: Interacts with users

## Technologies Used
- Arduino or Raspberry Pi microcontroller
- Sensors: Ultrasonic, IR, Load cell, Moisture for sorting
- Servo motors (lid mechanism)
- Display (LCD/OLED/Matrix)
- Speech module (e.g., TTS chip)
- Sanitizer pump (mini-peristaltic or solenoid)
